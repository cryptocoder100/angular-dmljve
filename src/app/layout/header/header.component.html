Keyboard legend

Global keyboard shortcuts:

  • Esc close popup
  • Alt + Up arrow close popup
  • Alt + Down arrow open popup
  • Arrow keys to navigate over the adjacent item
  • Alt + [_] shortcut for button is identified by the underscored character(if available)
  • Alt + Shift + [@] shortcut for verify email button that shows @
  • Alt + Shift + [<] shortcut for previous microdata button that shows < with the underscored character
  • Alt + Shift + [>] shortcut for next microdata button that shows > with the underscored character
  • Alt + [1] shortcut for dialog box answer "YES"
  • Alt + [2] shortcut for dialog box answer "NO"
  • Alt + [Any Number] shortcut for button is identified by the underscored character(if available)

Date picker specific keyboard navigation:

  • Alt + Down arrow open calendar
  • Alt + Up arrow close calendar
  • Ctrl + Left arrow navigate to previous month
  • Ctrl + Right arrow navigate to next month
  • Home highlight first day of the month
  • End highlight last day of the month

Home menu item navigation:

  • Alt + h highlight home on the navigation bar

Panel shifting shortcut:

  • Alt + 5 shift to next panel in round robin fashion(when applicable)

Quick access shortcut:

  • Alt + q shift focus to quick access input box (when applicable)

System setting shortcut:

  • Alt + a to enable/disable editing checkbox in system setting page
  • Alt + w shift focus to system settings tab

Scheduler specific keyboard navigation:

  • Entercreates new event from empty cell or selection. On selected event opens the edit popup window.
  • Tabmoves between available events.
  • Shift + Tabmoves between the available events backwards.
  • Esccloses the edit popup window.

Grid specific keyboard navigation:

Actions apply on grid header
  • Entersort by the column.
  • Alt + Down arrow open the filter menu
  • Escclose the filter menu.
Actions apply on grid data table
  • Page Uppages on previous page
  • Page Downpages on next page
  • Spaceselects the currently highlighted cell
  • Ctrl + Spaceselects or deselects the current cell, while persisting previously selected cells (only for selection mode "multiple")
  • Enteropen selected case (only available in case list page and find reporter page)

Tab specific keyboard navigation:

  • Arrow keys to navigate over the adjacent tab(When tabs are being highlighted)

Browse contol specific keyboard navigation:

  • Alt + l to get focus on browse (choose file) control on Import Data page.
  • Alt + m to get focus on second browse (choose file) control on Import Data page.
